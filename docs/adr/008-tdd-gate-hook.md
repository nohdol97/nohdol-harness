# ADR 008 — TDD 게이트 훅 (13절 실행 계층 강제)

> ⚠️ 이 중 "Claude Code 한정(Codex 미적용 수용)" 결정은 **ADR 014로 대체**되었고(git commit-msg 계층 추가로 도구 무관 강제), PreToolUse 등록·명령 파싱 계층 자체도 **ADR 015로 제거**되었다(git 계층 단일화). 스크립트 위치도 단일화 후속으로 `.agents/hooks/` → `.agents/githooks/`로 이동했다(2026-07-14 — hooks/는 Claude 세션 훅 전용, git이 부르는 것은 githooks/). 차단 지점(커밋 시점)·fail-open·예외 경로·CODE_EXTS 범위 결정은 유효하다.

- **날짜**: 2026-07-13
- **변경 내용**: 루트 `.claude/settings.json`을 신설하고 PreToolUse(Bash) 훅으로 `.agents/hooks/tdd-gate.py`를 등록한다. `git commit` 시점에 커밋 대상에 코드 파일이 있는데 테스트 파일 변경이 없으면 차단(exit 2)하고, 13절의 조치(테스트 동반 커밋 / 사용자 확인 후 `[no-test]` 표기)를 안내한다.
- **대상**: AGENTS.md 5·13절, `.claude/settings.json`(신설·추적), `.agents/hooks/tdd-gate.py`·`tdd-gate_test.py`(신설·추적), `docs/specs/2026-07-13-tdd-gate-hook.md`(스펙)
- **사유**: 13절 SDD+TDD는 문서 규칙이라 강제력이 없다 — 모델이 잊으면 그대로 통과된다. 인기 하네스(Superpowers 등)가 TDD를 훅으로 강제하는 패턴을 채택해, 문서 규칙에 실행 계층의 보증을 더한다(2026-07-13 사용자 승인).

## 결정

| 결정 | 내용 | 근거 |
|---|---|---|
| 차단 지점 | `git commit` 시점 (PreToolUse/Bash) | 13절의 완료 선언이 실체화되는 유일한 관문이 커밋이다. 파일 편집 시점 차단은 리팩토링 중간 상태까지 막아 마찰만 남는다 |
| fail-open | 판단 불가(JSON 파싱 실패, git 오류, 예외)는 전부 통과 | 훅은 보조 게이트고 규칙 원본은 문서다(ADR 006 "규칙의 운반체는 문서"). 훅이 정상 커밋을 막으면 우회를 학습시켜 게이트 자체가 죽는다 |
| 예외 경로 | ① 루트 하네스 저장소(문서 중심 — 5절 직커밋 예외와 동일 근거) ② `dev/`(실험 공간) ③ 최초 커밋(스캐폴딩) ④ 커밋 메시지 `[no-test]`(동작 불변 수정, **사용자 확인 전제**) — **이 4개가 전부이며, 환경변수 등 무추적 바이패스는 두지 않는다**(초안의 `TDD_SKIP=1`은 reviewer 1차 검증 F1로 제거 — 커밋 메시지에 흔적이 남지 않는 우회는 게이트를 무의미하게 한다) | 13절 적용 범위("기능 추가·동작 변경")와 일치시킨다. 예외 없는 게이트는 첫 마찰에서 통째로 꺼진다 |
| 경로 분기형 단일 훅 | 훅은 루트 settings 하나뿐이고, 스크립트가 대상 저장소 경로를 검사해 분기 | 12절 훅 규칙 — 훅은 세션 루트 것만 실행되므로 하위 프로젝트별 훅은 애초에 불가능 |
| Claude Code 한정 | Codex 세션에는 이 훅이 작동하지 않음을 수용 | 훅은 CLI별 편의 계층이다. 규칙은 13절(문서)이 운반하므로 Codex도 규칙 자체에는 구속된다 |
| settings.json 추적 | `.claude/settings.json`을 git에 올린다(5절 추적 목록에 추가) | 훅은 설치처별 데이터가 아니라 공용 규칙의 강제 장치다 — 모든 설치처에서 같아야 한다. 개인 설정은 settings.local.json(미추적)으로 분리 가능 |
| 스크립트 위치 | `.agents/hooks/` (추적, `.claude/` 아님) | 11절 — `.claude/`는 심링크 계층이라 실파일을 두지 않는다. settings.json은 12절이 명시적으로 루트 `.claude/` 위치를 규정한 유일한 예외 |
| 대상 확장자 범위 | CODE_EXTS는 앱 코드 중심 — `.sh`/`.sql`/`.tf` 등 스크립트·마이그레이션·IaC는 제외 | 테스트 관행이 낮은 파일군까지 막으면 게이트 마찰 > 가치 — 잦은 거짓 차단은 우회를 학습시킨다 |
