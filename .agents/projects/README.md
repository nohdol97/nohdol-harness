# .agents/projects/ — 하위 프로젝트 하네스 원본

하위 프로젝트 하네스의 **원본이자 유일본** 디렉토리.

- 구조: `.agents/projects/<이름>/` — `AGENTS.md`(도메인 규칙 + "스킬 후보" 섹션), `adr/`, 그리고 지연 생성되는 `skills/`·`agents/`. `<이름>`은 REGISTRY.md 레지스트리 행과 일치해야 한다.
- 하위 CLAUDE.md는 만들지 않는다(로더 없는 죽은 파일).
- REGISTRY.md처럼 **설치처별 데이터라 git 미추적**이다(이 README만 추적) — 컴퓨터마다 두는 프로젝트가 다르다.
- 스킬·에이전트는 **초기에 만들지 않는다** — 반복 작업 관찰·등록 가치 발견 시 사용자 승인으로 `skills/`·`agents/`에 생성하고, 하위 AGENTS.md에 목록·경로를 명시한다(CLI 자동 발견 불가 — 라우팅이 문서를 통해 로드).
- `project/<이름>/`에는 하네스 파일을 두지 않는다(심링크·복사본 포함).
- 상세 규칙: 루트 AGENTS.md 12절, docs/adr/006·007.
